# Prismic Custom Type Template

You can paste the custom type JSON template to the [custom type JSON editor](https://prismic.io/docs/core-concepts/content-modeling-with-json).

## Custom Type Setup
| Name        | API ID      | Type        |
| ----------- | ----------- | ----------- |
| Blog Post   | blog_post   | Repeatable  |
| Event       | event       | Repeatable  |
| Footer      | footer      | Single      |
| Header      | header      | Single      |
| Homepage    | homepage    | Single      |
| Project     | project     | Repeatable  |
