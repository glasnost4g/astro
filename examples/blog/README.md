# Astro Blog Example

Features:

- ✅ SEO-friendly setup with canonical URLs and OpenGraph data
- ✅ Full Markdown support
- ✅ RSS 2.0 generation
- ✅ Sitemap.xml generation

## Setup

```
yarn
```

## Dev

```
yarn start
```

Preview at `http://localhost:3000`

## Build

```
yarn build
```

Will output static site at `./dist`
